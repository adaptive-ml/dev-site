# RL Glossary

Adaptive's RL glossary.

Live at [dev.adaptive-ml.com](https://dev.adaptive-ml.com/).

## Setup

```bash
git clone https://github.com/adaptive-ml/dev-site.git
cd dev-site
bun install
bun run dev
```

## Build

```bash
bun run build
```

Deploys automatically to GitHub Pages on push to `main`.
