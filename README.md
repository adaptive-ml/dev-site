# RL Glossary

Adaptive's RL glossary — a structured reference that makes reinforcement learning for LLMs accessible.

Live at [adaptive-ml.github.io/dev-site](https://adaptive-ml.github.io/dev-site/).

## Setup

```bash
git clone --recurse-submodules https://github.com/adaptive-ml/dev-site.git
cd dev-site
bun install
bun run dev
```

## Build

```bash
bun run build
```

Deploys automatically to GitHub Pages on push to `main`.
