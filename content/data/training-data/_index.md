The raw text that [pretraining](/training/pretraining) runs on.

Petabytes of text, scraped from the internet, cleaned, assembled. The quality ceiling of any model is set here. The pipeline has four stages: [web crawl](/data/training-data/web-crawl) → [filtering](/data/training-data/filtering) → [deduplication](/data/training-data/deduplication) → [mixing](/data/training-data/mixing). Each step shapes what the model learns.
